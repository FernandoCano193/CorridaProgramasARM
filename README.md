### Ejecución del programa delayExample
El programa realiza operaciones AND y OR entre dos valores 11110000b y 10101010b. 
Los almacena en R0 y R1
|     | BIN      | HEX |
|-----|----------|-----|
| R2  | 11110000 | F0  |
| R3  | 10101010 | AA  |
| AND | 10100000 | A0  |
| OR  | 11111010 | FA  |
